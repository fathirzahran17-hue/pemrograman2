Project: pertemuan13
Materi: Pertemuan 13 - Aplikasi Web / Form Utama Aplikasi Administrasi Nilai Web

Cara menjalankan di NetBeans:
1. Extract ZIP.
2. Buka NetBeans.
3. File > Open Project.
4. Pilih folder pertemuan13.
5. Klik kanan project > Run.
6. Gunakan Apache Tomcat 9.

URL utama:
http://localhost:8080/pertemuan13/
